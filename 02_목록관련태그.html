① ul Tag : 순서없는 목록을 작성하는 (부모)Tag

② ol Tag : 순서있는 목록을 작성하는 (부모)Tag

③ li Tag : 목록에 들어가는 내용을 작성하는 (자식)Tag

            ★ ol Tag의 type 속성 ★
            type = "a" --> 영어 소문자 순서
            type = "A" --> 영어 대문자 순서
            type = "i" --> 로마 소문자 순서
            type = "I" --> 로마 대문자 순서

            + 추가속성
            start = "6" --> 시작하는 순서를 설정 (6부터 시작)
            reversed    --> 순서를 반대로
