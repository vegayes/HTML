글자 관련 기타 Tag

① b Tag : 단순히 글자를 굵게 표시하는 태그
② strong Tag : 글자를 강조하는 태그 (웹접근성) :: 음성으로 인식할 때, 강조해서 말해주는 역할
③ i Tag : 단순히 글자를 기울이는 Tag
④ em Tag : 글자를 강조하는 Tag / 기울이는 Tag(웹접근성)
⑤ mark Tag : 형광펜 효과를 나타내는 Tag
⑥ u Tag : 밑줄을 긋는 Tag
⑦ s Tag : 취소선을 긋는 Tag
⑦ del Tag : 취소선을 긋는 Tag2
⑧ small Tag : 글자를 작게
⑨ sup Tag (윗첨자) : x 2
⑩ sub Tag (아랫첨자) : log10
⑪ abbr Tag : 마우스 오버 시 툴팁 출력 Tag     HTTP
