영역을 나누는 형식(성질)

            
block 형식

- 공간을 수직 분할
- 크기를 지정하는 width, height 속성을 사용할 수 있음.

ex) p/pre (문단 나누기) , h1~h6 (제목 영역 나누기)
    hr (수평선 넣어서 화면 나누기), 
    div (영역 나누기)
        
            
 inline형식

- 공간을 수평 분할
- 크기를 지정하는 width, height 속성을 사용할 수 없음

ex) strong/b, em/i, mark, span(한줄을 나누는 용도)

block(div) / inline(span) 차이점 1: 줄바꿈(개행)

div Tag

block 형식의 Tag는 한 줄을 모두 차지하기 때문에
뒤에 오는 내용은 다음줄에 작성이 된다.

첫번째 영역
두번째 영역
세번째 영역

span Tag

inline 형식 Tag는 한 줄 내에서 일정한 칸을 차지하기 때문에
뒤에 오는 내용은 줄이 바뀌지 않고 옆에 붙는다.

첫번째 영역 두번째 영역 세번째 영역

block(div) / inline(span) 차이점 2: 영역 지정 방식

  • block : 사각형 박스 형태로 영역을 지정
  • inline : 내부에 작성된 내용(Content) 단위로 영역을 지정
