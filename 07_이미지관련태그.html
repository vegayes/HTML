img Tag 

            - 웹 문서에 사진, 그림 같은 이미지를 삽입하는 용도의 Tag
            
            [속성]
            1) src : 삽입할 이미지의 경로를 지정하는 속성(파일경로(상대경로),웹주소==이미지 주소 복사(절대경로))
            2) width / height : 이미지의 크기를 지정하는 속성 // 지정안하면 원본크기임. 
            3) alt : 이미지를 설명하는 속성 -> 이미지가 출력되지 않는 경우 화면에 직접 표시
                    (웹접근성)

src 속성 확인

width, height 속성을 이용한 크기 조절(고정/가변 크기)

① 고정 크기 단위(px) : 화면의 크기와 상관없이 지정된 값의 크기를 갖는 단위

② 가변 크기 단위(%) : 화면이나 영역 기준 사이트 크기에 따라 이미지 크기가 변경되는 단위 ==> 반응형!! (거의 필수) ex) 브라우저 크기의 퍼센트로


alt 속성 확인

이름이나 경로를 틀린경우에 사진을 파악할 수 없어서 alt속성으로 이동됨 즉, alt에 설정한 글자가 출력됨.
