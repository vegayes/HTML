04_표관련태그

            
① table Tag

            - 웹 문서에서 자료 정리를 위해 주로 사용하는 Tag
            - 행과 열로 이루어져 있으며, 행과 열이 교차하는 지점을 셀(cell)이라 함.
            
            -  table Tag는 테이블을 나타내는 행과 열이 작성될 수 있는 영역을 지정함. 
            ==> 최상위 부모 
            ==> border 설정 가능! 
        
            
② tr Tag

            - Table Row의 약자로 한 행을 나타내는 Tag
        
            
③ td Tag

            - Table Data의 약자로 한 행의 한 컬럼(==cell)을 나타내는 Tag
        
            
④ th Tag

            - Table header의 약자로 컬럼명을 표시하는 용도의 Tag
            - 기본 성질은 td와 같으나 추가적으로 굵은글씨, 가운데 정렬이 이루어짐.
        
            
⑤ caption Tag

            - 테이블의 제목이나, 설명 내용을 추가하는 Tag
