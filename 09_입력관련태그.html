input Tag

웹 문서에서 사용자가 입력할 수 있는 양식을 제공하는 태그

① text 관련 input Tag

1-1) type = "text" 

            - 한 줄짜리 문자열(텍스트)을 입력할 수 있는 입력 상자
            - input 태그의 type이 text로 기본값을 가지고 있어 생략가능!
            - inline형식(모든 input 태그가 inline)
        
            
★ text 관련 input Tag의 공용 속성

            - size : 입력 상자 크기 (px 단위)
            - maxlength : 입력 받는 텍스트의 최대길이
            - placeholder : 입력 상자의 작성될 내용을 설명 (글씨 입력시, 사라짐)

            * 내부에 작성된 input Tag의 값을 서버로 전달하는 역할
